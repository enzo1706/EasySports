## Uso de Skills

El agente cuenta con las siguientes skills disponibles:

### 1. find-skills (vercel-labs)
- Debes utilizar esta skill cuando necesites encontrar nuevas herramientas o capacidades.
- Es obligatoria cuando el usuario solicita funcionalidades no cubiertas por el proyecto actual.

### 2. skill-creator (anthropic)
- Debes utilizar esta skill cuando necesites crear herramientas personalizadas para el proyecto.
- Es obligatoria cuando la lógica requerida sea específica del dominio (partidos, asistencia, pagos).
